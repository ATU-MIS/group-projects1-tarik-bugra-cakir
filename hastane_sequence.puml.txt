@startuml
actor Patient
boundary "Randevu Ekranı" as UI
control "Sistem Mantığı" as System
database "Veritabanı" as DB

Patient -> UI: Randevu menüsünü açar
UI -> System: Klinik ve Doktor listesini ister
System -> DB: Listeyi getir
DB --> System: Klinik/Doktor listesi döner
System --> UI: Listeyi ekranda gösterir

Patient -> UI: Doktor ve Tarih seçer
UI -> System: Randevuyu kaydet (Pending)
System -> DB: Randevu bilgilerini yaz
DB --> System: Onay (Başarılı)
UI --> Patient: "Randevu Oluşturuldu" mesajı
@enduml
